### **Option 1: Quick Deployment (Recommended)**

1. **Go to:** https://vercel.com
2. **Sign in** with GitHub account
3. **For each project:**
   - Click "New Project"
   - Select `javascript-mini-projects` repository
   - Set **Root Directory** to project folder (e.g., `01-Guess-Number`)
   - Leave Build Command empty
   - Click "Deploy"
   - Copy the live URL
