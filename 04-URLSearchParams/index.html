4️⃣ What is URLSearchParams?

URLSearchParams is a built-in JavaScript Web API that makes it easy to read, add, update, and delete key-value pairs from the query string part of a URL (the part after the ?). It works like a simple dictionary for URL parameters — no manual string splitting needed.

Real-world use case: E-commerce websites use URL query parameters for filters — e.g. ?category=phones&brand=samsung&sort=price. When a user shares or bookmarks this URL, the same filters are applied again because the page reads the parameters using URLSearchParams on load.
